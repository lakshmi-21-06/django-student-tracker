# Student Performance Tracker

A Django web application for managing student records and grades.

## Features

- Add Student
- Edit Student
- Delete Student
- Add Grade
- Edit Grade
- Delete Grade
- Student Details
- Average Grade Calculation
- Subject Topper
- Class Average

## Technologies Used

- Python
- Django
- SQLite
- Bootstrap

## Run Locally

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
